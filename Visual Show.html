Pillar 1

🔀 Decomposition

Koi bhi bada problem direct solve karna mushkil hota hai. Decomposition matlab hai — bade problem ko chhote-chhote parts mein todna, taaki har part asaan lage.

🏠

Ghar banana

Seedha poora ghar mat banao. Pehle plan, phir neev, phir deewarein, phir chhat — ek-ek step.

📖

Exam ki taiyari

Pura syllabus ek saath nahi! Subject → Chapter → Topic → Points — tod ke padho.

🍕

Pizza banana

Dough, sauce, toppings, baking — alag alag steps mein toda jaye toh asaan ho jaata hai.

🎮 Try karo: Problem ko Todo!

Neeche apna problem type karo, phir uske parts (subproblems) add karo. Example: "Ghar saaf karna" → "Jhadu karna, Pocha karna, Bartan dhona"

Pillar 2

🔍 Pattern Recognition

Pattern Recognition matlab hai — repeating cheezein dhundhna. Jab ek baar pattern samajh aaya, toh usi solution ko doosri jagah bhi use kar sakte ho!

🌦️

Mausam

Badal kaale hote hain, hawa thand hoti hai — pattern pehchana, toh barish aane waali hai!

📐

Maths Patterns

2, 4, 6, 8… pattern samjho toh agla number easily bata sako. Even numbers ka pattern!

🔤

Language

"Cat, bat, rat, mat" — "at" pattern recognize karo, toh nayi words bhi padhna asaan!

🎮 Pattern Pehchano!

Sequence dekho aur "?" ki jagah sahi number/symbol choose karo.

Pillar 3

🎯 Abstraction

Abstraction matlab — zaruri cheezein rakhna, bakwas ignore karna. Jab problem solve karo, tab sirf important details pe focus karo.

🗺️

Map

Map mein har ghar ki bakwas nahi hoti — sirf raaste aur important jagahein. Abstraction!

📱

App Icons

WhatsApp icon sirf ek phone + bubble hai. Inside mein lakhon lines of code hain — par tum sirf icon dekhte ho.

🚗

Car Drive karna

Tumhe engine mechanics nahi chahiye — sirf steering, brakes, accelerator. Baaki abstracted hai!

🎮 Zaruri vs Bekaar — Chhanto!

Neeche ek scenario hai. Har detail ke saath decide karo: kya yeh problem solve karne ke liye zaruri hai, ya ignore karna chahiye?

Pillar 4

📋 Algorithm

Algorithm matlab ek clear step-by-step plan — jaise recipe. Agar steps sahi order mein ho, toh result hamesha same aayega!

🧁

Cake Recipe

Ingredients → Mix → Bake → Cool → Decorate. Yahi order follow karo, perfect cake milega.

🏧

ATM se Paisa

Card daalo → PIN dalo → Amount daalo → Confirm karo → Paisa lo. Yeh algorithm hai!

🎮

Game Logic

Player jumps → Check collision → Add score → Next level. Games sab algorithms se chaltey hain.
